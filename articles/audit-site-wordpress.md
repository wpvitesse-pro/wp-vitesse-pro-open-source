# Analyseur d’Audit WordPress

Ce script permet d’analyser 10 points critiques d’un site WordPress :
- Performance
- SEO
- Sécurité
- Accessibilité
- Mobile-Friendly
- Contenu
- Technologies
- Migration
- Plugins
- Score SEO d’entité

Retrouvez l’outil complet sur : https://wp-vitesse-pro.fr/audit-site
